# python 선행학습

## 개념정리
> 변수와 자료형
>
> 변수(Variable)란 데이터를 저장하는 공간이다. 변수를 사용하면 특정 값을 이름으로 저장하고 이후에 쉽게 참조할 수 있다.

- int : 정수형  
  - 숫자를 표현하는 자료형 
  - 예) 10, -5, 0
- float : 실수형 
  - 숫자를 표현하는 자료형
  - 예) 3.14, -0.5, 2.0
- str : 문자열 
  - 문자들의 집합이며, 작은따옴표 또는 큰따옴표로 감싸서 표현
  - 예) 'hello', 'world'
- bool : 논리형 
  - True 또는 False 값을 가질 수 있으며 조건문에서 주로 사용
  - 예) True, False
- list : 리스트
  - 여러 값을 저장할 수 있으며 순서가 있고 변경 가능
  - 예) [1, 2, 3, 4, 5]
- tuple : 튜플
  - 리스트와 비슷하지만 한 번 생성하면 수정할 수 없음
  - 예) (1, 2, 3, 4, 5)
- dict : 딕셔너리
  - 키-값(key-value) 쌍으로 데이터를 저장하는 자료형
  - 예) person = {"name": "Minseok", "age": 25}



> if, for, while
>
> 조건문은 특정 조건을 만족할 때만 코드를 수행하는 문법이다.

### 조건문
### if
- if문은 입력한 조건을 만족하는 경우에만 다음 문장을 수행하는 구문이다.
- if [조건] : (수행할 코드)
 
### 반복문
### for
- for [변수] in [반복가능한 객체]: [수행할 코드]
- 반복가능한 객체는 리스트, 튜플, 문자열 등이 있다.
- [반복가능한 객체]의 원소를 순서대로 순회하면서 하나씩 [변수]에 넣어준 뒤 [수행할 코드]를 수행하는 방식이다. 따라서 [반복가능한 객체]의 원소의 개수만큼 반복문을 수행하게 된다.

### while
- for문이 아닌 while문이라는 방식으로도 반복문을 수행할 수 있다.
- while [조건문]: [수행할 코드]
- [조건문]이 참인 경우에는 [수행할 코드]를 수행하고 거짓인 경우에는 while문을 빠져나감으로써 반복문을 종료한다.
